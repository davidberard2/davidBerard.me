    This project is a Android Application created for a Software Process class at Concordia University. With a team of 8 people, we build a program that mimics a Bulletin Board to sale items. It provides services similar to Ebay and Kijiji. We implemented features such as creating and modifying a profil, adding an item, searching by price or category, and creating an favorites list.
    I learned a lot about how to properly manage a project, motivate a team and communicate a vision. We learned to ducument the information, build numenclature and establish adapted process to solve issues. Another aspect of this project allowed me to understand the use to choose who-does-what in function of people's skills.
